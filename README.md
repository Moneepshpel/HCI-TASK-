## About the Project
This project is a UI/UX redesign of selected screens from the Spotify mobile application. The redesign focuses on improving usability, layout clarity, and visual hierarchy while keeping Spotify’s original brand identity and color scheme.

## Tools & Technologies
- Figma
- UI/UX Design Principles

## Project Scope
- Redesigning selected Spotify mobile app screens
- Creating a high-fidelity prototype
- No backend or frontend implementation

## How to View the Design
The redesigned Spotify screens are included in the project presentation.  
Open the presentation file in this repository to view all redesigned screens and explanations.
